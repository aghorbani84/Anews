ANews

A modern, full-stack news application with a Google News-inspired UI. Built with Flutter & FastAPI. Supports Dark Mode, RTL/LTR, and real-time news fetching.

View on GitHub

Powerful Features

📱

Modern UI

Beautiful Material 3 design with Breaking News carousel, category tabs, and smooth animations.

🌐

Bilingual Support

Instantly switch between English and Farsi. Full RTL layout support for a native experience.

🌙

Dark & Light Mode

Seamless Light and Dark theme support that follows system settings or user preference.

⚡

FastAPI Backend

High-performance asynchronous Python backend to fetch and serve news dynamically.

🔒

Secure Architecture

API keys are safely stored using environment variables (.env) and never exposed to the client.

🔄

Real-time Data

Fetches real news data using Webz.io API with robust error handling and pull-to-refresh.
